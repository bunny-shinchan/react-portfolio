
# Sid – Developer Portfolio

A modern personal developer portfolio built with **React + Vite**.
This portfolio showcases my projects, skills, experience, and provides a way for visitors to contact me.

The design focuses on a **clean developer aesthetic**, smooth animations, and responsive layout.

- Clean modern UI
- Responsive design (desktop, tablet, mobile)
- Smooth scroll navigation
- Animated sections using Framer Motion
- Project showcase cards
- Skills section
- Contact form with EmailJS integration
